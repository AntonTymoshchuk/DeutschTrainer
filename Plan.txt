Типы тренировок:

1. Зрительный (русский) - Вербальный (немецкий)
2. Зрительный (русский) - Письменный (немецкий)

3. Слуховой (русский) - Вербальный (немецкий)
4. Слуховой (русский) - Письменный (немецкий)

5. Зрительный (немецкий) - Вербальный (русский)
6. Зрительный (немецикй) - Письменный (русский)

7. Слуховой (немецкий) - Вербальный (русский)
8. Слуховой (немецкий) - Письменный (русский)

База данных MySql:

Таблица "Словарь":
id,
Слово по-немецки,
Слово по-русски,
Путь к звуковому файлу немецкого слова.
Путь к звуковому файлу русскоо слова.

Таблица "Оценка тренировки "Зрительный (русский) - Вербальный (немецкий)":
id (пары в словаре),
Количество правильных ответов,
Количество неправильных ответов,
Количество ответов "не знаю".

... И так со всеми типами тренеровок ...

Алгоритм подбора пар слов для каждой тренировки:
1. Одна копия оценок сортируется по количеству попыток (кол. правльных + кол. непрвильных + кол. "не знаю") по возрастанию.
2. Другая копия оценок сортируется по количеству неудачных попыток (кол. неправильных + кол. "не знаю") по спаданию.
3. Затем получившиеся списки складываются (елемент первого списка -> елемент второго списка -> ...).
4. Проверяется есть ли совпадения соседних элементв, эсти есть, то дубликат удаляется из списка.

После того как я ответил на одно задание любого типа тренировки:
1. Если я ответил правильно, то:
   1. кол. правельных ответов += 1.
   2. в список правильных ответов добавляется id этой оценки
   3. предоставляется следующее задание.
2. Если я ответил не правильно, то:
   1. кол. не правельных ответов += 1.
   2. показывается или прослушивается правильный ответ.
   3. Алгоритм подбора пар слов для каждой тренировки запускается заново, но из него удаляются все елементы, которые были правильны только что.
   4. предоставляется следующее задание.
3. Если я ответил, что "не знаю", то:
   1. кол. не правельных ответов += 1.
   2. показывается или прослушивается правильный ответ.
   3. Алгоритм подбора пар слов для каждой тренировки запускается заново, но из него удаляются все елементы, которые были правильны только что.
   4. предоставляется следующее задание.

После постановки задания даётся 30 секунд и 1 попытка правильно ответить,
если я отвечаю неправильно, программа показывает правильный ответ, если 30 секунд
исчерпаны, программа засчитывает мне ответ "не знаю".

При входе в программу есть 2 выбора: добавить слова в словарь, или тренироватся.

Если добавить слово в словарь, то есть поле ввода немецкого слова, поле ввода русского слова,
и кнопка "добавить". При нажатии на эту кнопку создаются записи немецкого и русского слов,
и сохраняются в файлы, затем все данные помещаются в таблицу "Словарь". Потом id этой записи
помещаются в каждую таблицу тренировок со значениями 0 0 0.

Если выбрано тренироватся, то выдается список тренировок.
